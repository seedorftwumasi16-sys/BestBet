# BestBet

Premium sportsbook platform — **Bet Smarter. Win Bigger.**

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **Framer Motion** (animations)
- **Recharts** (dashboard charts)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Premium sportsbook homepage with 3-column layout
- Live betting with real-time odds simulation
- Bet slip (single/multi) with booking codes
- User dashboard with charts and transaction history
- Enterprise admin panel
- Dark/light mode toggle
- Mobile-first with bottom navigation and floating bet slip
- PWA support with manifest and app icons
- Animated splash screen
- Full BestBet branding system
